Inventory-Optimization-Risk-Dashboard-H1-2026
Author: Rahima Akter Diti Role: Lead Supply Chain Data Analyst

This repository contains a strategic analysis and interactive dashboard for Amazon's H1 2026 inventory performance. It integrates Q1 2026 actuals with modeled H1 guidance to provide a holistic view of inventory risk, stockout exposure, and fulfillment efficiency.

Dashboard & Analysis Highlights
Financial Performance: Visualization of total H1 Revenue, Contribution Margins, and regional growth trends.

Strategic Focus: Analysis of stockout risks, overstock capital immobilization, and fulfillment center bottlenecks.

Interactive Data: Dynamic views allowing users to explore branch-level performance, dwell times, and risk status.

Tools & Techniques
Financial Modeling: Microsoft Excel (Q1 actuals + H1 projections).

Data Visualization: HTML, CSS, and data-driven reporting.

Disclaimer
This analysis is based on available Q1 2026 financial data and independent modeling. It does not represent internal Amazon operational data or official company forecasts.
